# WeatherEye Web Interface 🌤️

Веб-интерфейс для приложения погоды WeatherEye.

## Установка и запуск

### 1. Убедитесь, что у вас есть все файлы:
- `weather.py` - ваше CLI приложение
- `web_app.py` - веб-сервер
- `templates/index.html` - веб-страница
- `requirements-web.txt` - зависимости

### 2. Установите зависимости:
```bash
pip install -r requirements-web.txt
